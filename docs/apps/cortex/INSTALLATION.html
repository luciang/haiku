installing
add-ons

Sample add-ons are provided in the add-ons.ppc and add-ons.x86 folders. To install add-ons for your architecture, select all of the '.media_addon' files in the appropriate folder and drag them to the linked folder named '[DRAG HERE TO INSTALL]'.
