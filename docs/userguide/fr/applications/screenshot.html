Screenshot

Deskbar :Applications
Localisation :/boot/system/apps/Screenshot
Réglages :~/config/settings/screenshot


En plus de la copie d’écran classique via la touche IMPR ÉCRAN, qui crée une image PNG de l’écran dans /boot/home/, cette application fournit quelques options supplémentaires.

Démarrer Screenshot fera une capture d'écran comme d'habitude, mais vous permettra de contrôler depuis la fenêtre principale, le nom, le format et le dossier du fichier avant d'enregistrer l'image.

La boite de dialogue Options contient quelques paramètres de plus :

En plus de permettre le choix entre une capture de l’écran entier (Capture entire screen) ou de la fenêtre active (Capture active window), d’inclure ou non la bordure de la fenêtre dans ce second cas (Include window border), cette fenêtre vous permet aussi d’entrer un délai avant la capture. Bien sûr, ce délai n’est effectif que si la capture est prise en pressant le bouton Take Screenshot.

Effectuer un capture d'écran depuis le Terminal

L'application Screenshot est aussi utilisable depuis le Terminal ou depuis un script.
Screenshot --help affiche les options en paramètres :

~> Screenshot --help

Screenshot [OPTION]... Creates a bitmap of the current screen

OPTION
  -o, --options         Show options window first
  -m, --mouse-pointer   Include the mouse pointer
  -b, --border          Include the window border
  -w, --window          Capture the active window instead of the entire screen
  -d, --delay=seconds   Take screenshot after specified delay [in seconds]
  -s, --silent          Saves the screenshot without showing the app window
                        overrides --options, saves to home folder as png

Note: OPTION -b, --border takes only effect when used with -w, --window
