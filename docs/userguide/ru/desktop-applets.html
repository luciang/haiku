Приложеница для рабочего стола

Haiku предлагает несколько полезных инструментов, которые могут быть установлены на Рабочий стол или Deskbar посредством их репликантов.

LaunchBox  Ярлыки для ваших любимых приложений.
NetworkStatus  Отображает статус подключения.
ProcessController  Наблюдайте и управляйте каждым запущенным приложением или службой.
Workspaces  Миниатюрная версия всех рабочих столов.
