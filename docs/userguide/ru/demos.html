Демонстрационные приложения

Существует несколько маленьких игры и демонстрационных приложений для вашего развлечения. Большинство демонстрационных приложений ориентированы на разработчиков, которые заинтересованы в изучении кода эти приложений. В Haiku весь исходный код открыт!

BSnow  Пускает снег по всему рабочему столу.
Chart  Позволяет путешествовать сквозь звезды и галактики.
Clock  Аналоговые часы.
Cortex  Показывает media_server во всей красе.
FontDemo  Слайд-шоу всех ваших шрифтов.
GLDirectMode  Демонстрация OpenGL.
GLTeapot  Демонстрирует OpenGL, вращающийся чайник.
Gradients  Демонстрация API градиентов.
Mandelbrot  Вычисление фрактолов.
Pairs  Игра, суть которой заключается в поиске парных изображений.
Playground  Демонстрация API рисования.
Pulse  Отображает нагрузку ЦП.
Sudoku  "Солитер" этого века.
