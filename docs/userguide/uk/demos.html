Демки (Demos)

Тут є кілька маленьких гарних ігор і демо додатків для вашої розваги. Більшість з них призначені показати розробникам можливості коду цих додатків, які як і вся Haiku є з повністю відкритим кодом.

BSnow  Просто падає сніг!
Chart  Мандруємо серед зірок і галактик.
Clock  Аналоговий годинник.
Cortex  Показує media_server у всій красі.
FontDemo  Слайд-шоу всіх Ваших шрифтів.
GLDirectMode  Демонстрація OpenGL.
GLTeapot  Обертання чайника оброблене при допомозі OpenGL .
Gradients  Демонстрація градієнтів API .
Mandelbrot  Обчислення фрактолів.
Pairs  Гра з пошуку пар одинакових зображень.
Playground  Демонстрація API малювання.
Pulse  Показує завантаженість процесора Вашого ком'пютера.
Sudoku  "Solitair" цього сторіччя.
