Network

Deskbar:Preferencias
Ubicación:/boot/system/preferences/Network
Configuración:/boot/common/settings/network/resolv.conf


En la parte superior se selecciona el adaptador de red que se va a configurar.

A continuación se especifica si la red se obtiene automáticamente (vía DHCP) o usando una dirección estática. En caso de utilizar una dirección estática, hay que llenar los campos IP Address, Netmask, Gateway y DNS Servers manualmente. De lo contario, el panel mostrará las direcciones actualmente seleccionadas con DHCP.

Revert Revierte las configuraciones al estado en que estaban activas cuando se inició el panel de preferencias Network.
Apply Aplica la configuración ingresada.
