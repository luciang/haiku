Screenshot

Deskbar:Aplicaciones
Ubicación:/boot/system/apps/Screenshot
Configuración:~/config/settings/screenshot


Además de las capturas de pantalla tomadas mediante la tecla Impr Pant, que coloca un PNG de la pantalla actual dentro de /boot/home/, esta aplicación proporciona algunas opciones útiles.

Al iniciar Screenshot se tomará una captura de pantalla como siempre, pero su ventana principal ofrece controles para selecionar el nombre de archivo, formato y carpeta dónde salvar la imagen.

El diálogo Opciones tiene unas cuantas selecciones adicionales:

Además de lo obvio, cambiar entre tomar toda la pantalla o sólo la ventana activa, inluyendo la ventana activa, incluir los bordes de las ventanas y el puntero del ratón, puede ingresar una demora antes que se tome la captura de pantalla. Por supuesto, esta demora sólo tiene efecto si toma una nueva captura al presionar Take Screenshot (Tomar captura de pantalla).

Tomar una captura de pantalla desde la Terminal

La aplicación de captura de pantalla también se puede usar desde la Terminal o un script.
Screenshot --help muestra las opciones familiares como parámetros:

~> Screenshot --help

Screenshot [OPTION]... Creates a bitmap of the current screen

OPTION
  -o, --options         Show options window first
  -m, --mouse-pointer   Include the mouse pointer
  -b, --border          Include the window border
  -w, --window          Capture the active window instead of the entire screen
  -d, --delay=seconds   Take screenshot after specified delay [in seconds]
  -s, --silent          Saves the screenshot without showing the app window
                        overrides --options, saves to home folder as png

Note: OPTION -b, --border takes only effect when used with -w, --window
