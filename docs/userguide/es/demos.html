Demostraciones

Existen algunos pequeños juegos y aplicaciones de demostración para su entretenimiento. La mayor parte de las demostraciones están enfocadas a desarrolladores que estén interesados en aprender del código, el cual es código abierto como todo en Haiku.

BSnow  Deje que nieve.
Chart  Moverse a través de estrellas y galaxias.
Clock  Un reloj analógico.
Cortex  Visualizar el media_server funcionando.
FontDemo  Un pase de diapositivas de todos sus tipos de letra.
GLDirectMode  Una demostración de OpenGL.
GLTeapot  La tetera giratoria renderizada por OpenGL.
Gradients  Una demostración de la API de gradientes.
Mandelbrot  Cálculo de fractales.
Pairs  Un juego de encontrar parejas de iconos iguales.
Playground  Probar la API de dibujo.
Pulse  Mirar la carga de trabajo de la CPU.
Sudoku  El "solitario" de este siglo.
